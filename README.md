# HaixinSkills — 内容创作技能库

可复用的工作流技能集合，每个技能一个文件夹，包含 `SKILL.md`（核心流程）+ 必要时参考文件。

## 技能列表

### 1. warm-real-story-collection — 暖心人间故事搜集
搜集/原创短篇暖心真实故事：普通人面对困境时的尊严、邻里间不声张的善意。
详见 `warm-real-story-collection/SKILL.md`

### 2. warm-real-story-video — 暖心故事视频制作
从故事到视频的完整 SOP：拆解叙事旁白 → AI 生图带文字 → 验证 → B&W→彩色擦除视频组装。
详见 `warm-real-story-video/SKILL.md`

### 3. make-tutorial-video — TikTok 教程视频制作
制作竖屏 App 教程视频：Demo 效果展示 → 屏幕录像 → TTS 解说 → 逐词高亮字幕。7 步 SOP + 模板脚本 + Remotion 字幕模板。
详见 `make-tutorial-video/SKILL.md`
