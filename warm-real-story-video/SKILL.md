# 暖心故事视频制作 — 完整 SOP

从故事到成片，五阶段全流程。

---

## 全流程概览

```
Phase 0 ──→ 输出故事列表，用户选一篇
Phase 1 ──→ 用户选风格 → 输出叙事脚本（图数 + 每图画面描述 + 每图字幕）
Phase 2 ──→ 用户确认脚本 → 逐张生图（dream_nano_banana_pro）
Phase 3 ──→ 拼接视频（白底衬板 9:16 + 交叉淡化 + BGM 铺底）
Phase 4 ──→ 输出三套标题/简介/Hashtag，供挑选
```

---

## Phase 1 — 风格选择 + 叙事脚本

### 1.1 可选风格

从 https://github.com/threerocks/hand-drawn-styles 即时 fetch 最新配方。

| # | 风格 | 适合场景 |
|---|------|----------|
| 1 | 纯人类手绘儿童涂色页 | 温情、天真、亲子共鸣 |
| 1.1 | 儿童涂色页-低饱和克制版 | 克制沉重叙事（如离世主题） |
| 2 | 极简黑白线条讲解漫画（xkcd 火柴人） | 轻量、幽默、解说 |
| 3 | 蜡笔童涂（5岁小孩坏画） | 更粗糙、更原生 |
| 3.1 | 蜡笔童涂-潦草自画版 | 纯自画像感 |
| 4 | 吉卜力风 | 唯美童话 |
| 5 | 小豆人黑色涂鸦信息图 | 图解/科普 |
| 6 | MS Paint 烂涂鸦 | 故意丑、网感 |
| 7 | 圆珠笔单线涂鸦 | 速写感、粗糙 |
| 8 | 蜡笔实拍（真·儿童手涂纸张） | 最真实的儿童画感 |
| 9 | 水墨写意 | 诗意、留白 |
| 10 | 复古像素 | 像素游戏风 |
| 11 | 情绪叙事淡彩速写 | 水彩淡彩、情绪氛围 |
| 12 | 复古动画概念稿 | 动画设定感 |

**推荐**：暖心故事首选 **#1 纯人类手绘儿童涂色页**。

### 1.2 确认：图数 + 风格

- 默认 10 张图
- 风格确认后，fetch 对应风格的完整 prompt 模板
- 每张图尺寸 3:4（896×1200），最终视频 9:16（1080×1920）

### 1.3 输出叙事脚本

对用户确认的故事拆解为 10 个画面，每画面输出：

```
图 N | 字幕文字
画面描述（一句中文，说清谁在哪做什么，不超 40 字）
```

字幕文字就是最后图片下方会出现的手写文字，必须精简、有叙事推进力。

### 1.4 输出生图 prompt

把画面描述嵌入风格 prompt 模板，确保：
- 风格描述在 **前**
- 字幕指定在 **中**：`画面下方必须有清楚可读的粗黑手写中文字："<字幕>"`
- 画面描述在 **后**：`画面内容：<画面描述>`

→ 用户确认后进入 Phase 2。

## Phase 2 — 生图

逐张调用 `dream_nano_banana_pro`，参数固定：
- `aspectRatio`: `"3:4"`
- `imageSize`: `"1K"`
- `save_as`: `"<故事名>_N_<代号>"`
- `text`: 完整 prompt（风格模板 + 字幕指定 + 画面描述）

生图异步提交（fire-and-forget），12 张约需等待回调。全部完成后检查文字可读性（抽查首尾各 2 张）。

**文字不合格则重生成单张。**

---

## Phase 3 — 拼接视频

### 3.1 整理图片

```bash
mkdir <故事名>/imgs
move/rename 所有生图 → imgs/img_01.jpeg ~ img_N.jpeg
```

### 3.2 参数

| 参数 | 值 |
|------|-----|
| 目标尺寸 | 1080×1920（9:16 竖屏） |
| 每图展示 | 4.5s |
| 交叉淡化 | 0.5s fade |
| FPS | 24 |
| BGM | `.manify/runs/<sessionID>/ref_bgm.mp3`，音量 0.35 |
| 总时长 | 约 48-50s（10 张） |

### 3.3 图片预处理

**白底衬板**（不裁切）：宽度撑满 1080，高度等比缩放，居中放在 1080×1920 白画布上。上下自然留白。

### 3.4 编码

ffmpeg xfade 滤镜链交叉淡化，叠加 BGM。生成 `<故事名>_final.mp4`。

### 3.5 B&W→彩色擦除过渡（可选进阶）

原始 `build_story_video.py` 支持的 B&W→彩色划像过渡：先用 PIL 将每张图处理成 B&W 渐变版本，然后用 `overlay` + `xfade` 做 wipe 过渡。仅在用户明确要求时使用。

## Phase 4 — 发布元数据

输出三套标题/简介/Hashtag，供操作员挑选。三套在叙事角度和节奏上拉开差异。

### 方案 A：一句话克制型
像电影海报上的那句话，留白多，不解释。
- 标题 ≤10 字，有画面感
- 简介：一句讲清核心，不加评论
- Hashtag：核心话题 3-4 个 + 流量话题 2-3 个

### 方案 B：细节钩子型
用具体细节吸引注意力，制造"然后呢？"
- 标题：带一个具体细节钩子
- 简介：2-3 句，先制造冲突或反差，再反转
- Hashtag：情绪共鸣和生活方式词

### 方案 C：概括叙事型
用一段话把完整故事讲出来，信息量最大。
- 标题：中性概括
- 简介：完整段落起承转合，没看视频也能读懂
- Hashtag：核心话题 + 情感词

## 格式示例（驿站的写字板）

### 方案 A
- **标题**：驿站那块写字板
- **简介**：聋哑妈妈带着三岁女儿在驿站打工。一块小白板，成了她和整个小区之间的桥梁。
- **Hashtag**：#聋哑妈妈 #快递驿站 #人间温暖 #被陌生人暖到的瞬间 #治愈系故事

### 方案 B
- **标题**：小区快递驿站来了个聋哑店员，有人放了一块白板
- **简介**：第一天被人吼，比划半天看不懂取件码。第二天柜台上多了一块白板。一个月后上面写着"你女儿好可爱""下雨了记得带伞"。
- **Hashtag**：#聋哑妈妈 #快递驿站 #被生活温柔以待 #平凡人的不平凡 #治愈 #真实故事

### 方案 C
- **标题**：驿站有个说不了话的妈妈
- **简介**：楼下的快递驿站，老板娘是聋哑人，带着三岁的女儿。起初有人不耐烦，后来不知道谁在柜台上放了一块小白板。取快递的人开始在上面留字——"今天辛苦啦""你女儿真可爱""下雨了记得带伞"。她看完就笑，擦干净放回原处。一块板子，让一个小区的人学会了另一种说话。
- **Hashtag**：#聋哑妈妈 #快递驿站 #人间温暖 #治愈系故事 #被陌生人暖到的瞬间

## 注意事项

1. **先列故事再做事**：不要直接开始生图，必须先让用户选故事
2. **prompt 中的文字在画面内**：字幕由 AI 直接在图中画出，不是后期叠加
3. **白底衬板不裁切**：3:4 图进 9:16 画布，必须 pad 不 crop
4. **生图后校验文字**：抽查首尾各 2 张，确认手写文字可读
5. **不要生成后再补 Phase 0/1**：Phase 0 和 1 在生图之前完成
6. **三套元数据在视频完成后输出**：不是视频之前
